## Installation Commands

### 2026-02-16

- `npm init` Initialized as NPM package and created package.json
- `npm install express` Node.js web application framework
- `npm install -D nodemon` Hot reload for every changes on files
- `npm install chalk` Add Colors to logs
- `npm install mongodb` Database for permanent data storage
- `npm install -D dotenv` Environment secrets

### 2026-02-18

- `npm install sanitize-html` Strip off any HTML from the user input value
